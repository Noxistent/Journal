

## Стек технологий
* **UI:** Jetpack Compose
* **Network:** Retrofit + Gson
* **Storage:** Tencent MMKV (высокопроизводительное хранилище)

## Лицензия проекта
Данный проект распространяется под лицензией **GNU GPL v3.0**. Полный текст лицензии находится в файле [LICENSE](./LICENSE).
